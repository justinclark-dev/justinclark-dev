About this Project

I have experience using MongoDB, Express.js, and Node.js, therefore this project is helping me learn Angular so that I can develop using the full MEAN stack. Angular with it's CLI tool, is a great for building a front-end, but the one thing that I don't like how it downloads the full framework into the node_modules folder. It would be nice Angular could auto-load just the parts you need.

This project is hosted in Heroku where I have set up a pipeline with automatic build deployments with git pushes via the terminal. Heroku is Awesome!
