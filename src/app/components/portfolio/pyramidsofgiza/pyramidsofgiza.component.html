The Great Pyramids of Giza

An image graphic I created solely with Python code.

*No webpage, graphic is created with a script. View code by following GitHub link at bottom of the page.

About this Project

My task for this assignment was to use Python code build a scene with three different repeating scene elements of varying sizes. Each element had to be composed of at least two shapes, and each element had to be created by calling the same function. My three repeating scene elements where: pyramids, clouds, and birds.

For the pyramids I used two triangles, one side with the appearance of sunshine (hence the sun in the upper-left corner), and the other with shadows by using slightly different shades to give the appearance of distance.

I gave more depth to the image by making the clouds smaller, darker, and lower as they progress to the right of the scene. The clouds where created with overlapping circles of varying sizes. I put light colored circles in front of black circles to make the top shadows, and a sky colored rectangle with a top black border to hide the bottoms of the circles to make the flat bottom of the clouds.

I used the same overlapping circles method to create the birds and also the moon. For each bird, I used color of the background the bird was flying in front of to create the front circles.
