About this Project

This was originally a class assignment to build a simple home page that navigates to second page.

I'm currently building it into a full front-end experience to showcase my design and development skills.

Being new to the "mobile first" design concept, I totally built the current draft with the desktop experience first out of habit. Now I totally get why it's much easier to do mobile first design. I'll be rewriting the css with mobile first, so responsiveness will be coming soon.
